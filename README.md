# day-01 (2021-02-21)
- 상수 선언 === let 상수명: 타입 = 값 
- 변수 선언 === var 변수명: 타입 = 값 
---
### swift 기본(primitive)타입 
 - Bool : true, false
 - Int : 정수
 - UInt : 양의 정수
 - Float : 실수 32비트 
 - Double : 실수 64비트 
 - Chracter
 - String
 기본적으로 swift는 서로 다른 타입과 자료 교환은 암시적으로 불가능 
 ---
 ### Any, AnyObject, nil 
  - Any : 모든 타입을 지칭하는 키워드 
  - AnyObject : 모든 클래스 타입을 지칭하는 프로토콜
  - nil : 없음을 의미하는 키워드
